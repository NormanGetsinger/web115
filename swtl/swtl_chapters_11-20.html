Chapter 15: Arrays

An array is a variable with a list of values

var sushiSelection = ["Tuna Sushi", "Salmon Sushi", "Lobster Sushi"];

Chapter 16: Arrays: adding and removing elements

This chapter talks about arrays and how to add or remove elements from the array

The keywords are push and pop. Pop removes the last item from an array and push adds to the last item in an array

var sushiSelection = ["Tuna Sushi", "Salmon Sushi", "Lobster Sushi"];

sushiSelection.push("Eel Sushi"); would add "Eel Sushi" to the end of the sushiSelection array.

Chapter 17: Arrays: removing, inserting, and extracting elements

The key words here are:

  • Shift: remove an element from the beginning of an array
  • Unshift: add an element from the beginning of an array
  • Splice: add one or more elements to an array
  • Slice: copies elements from an array to another array

var sushiSelection = ["Tuna Sushi", "Salmon Sushi", "Lobster Sushi"];

sushiSelection.shift(); would remove "Tuna Sushi" from the sushiSelection array.
