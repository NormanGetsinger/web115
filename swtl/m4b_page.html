Chapter 1: Alerts

Alerts are things that a user will see when visiting a website or clicking on a button

input: alert("Welcome to Mr. G's asian kitchen!");

output: Welcome to Mr.G's asian kitchen!

Chapter 2: Variables for strings

A variable is a string if there are quotes around it.

input: var name = "John:;

output: console.log(name); will return John

Chapter 3: Variables for Numbers

Variables can be numbers, so long as quotes are not around the number

input:var salmonRoll = 9.50;, var tunaSashimi = 3;, total = salmonRoll + tunaSashimi;

console.log(total); output: 12.5

Chapter 4: Variable Names Legal and Illegal

  1. You cannot use a space for a variable name, an example would be var sushi roll.
  2. You can't use reserved words for variable names in Javascript.
  3. Legal characters that are not alphabet characters or numbers are _ and $.
  4. All variables should be written using camelCasing. var salmonRoll is fine, var salmonroll is not.

Chapter 5: Math Expressions: familiar operators

These are operators that we have been using our entire life and they are very useful in Javascript

  • * is multiplication
  • / is division
  • + is addition
  • - is subtraction
  • % is modulus

Chapter 6: Math Expressions: unfamiliar operators

These are expressions that can increase or decrease a variable by one. The operators are ++ and --.

x = 5

x++

console.log(x); this will return 6

Chapter 7: Math Expressions: eliminating ambiguity

This is all about using paranthesis to control the order of operation.

An example would be (3*4)-(2+6) will equal 4

Chapter 8: Concatenating text strings

You would use the + operator to put strings together.

An example might be var thankYouMessage = "Thank you for your " + "purchase, " + "Mike."

You would then get the message, Thank you for your purchase, Mike.

Chapter 9: Prompts

A prompt is similar to an alert, but the user is able to type something into the box.

An example would be prompt("What would you like to order?");

Chapter 10: if statements

If statements test your code to see if something is true.

An example would be, if(purchase === "phoOrderB"){
purchase = "Steak Pho";
}
