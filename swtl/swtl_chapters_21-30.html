SWTL 21-30


Chapter 21: Changing case

You can change the case of a string to upper or lower case in Javascript

input: var pet = "DOG";

console.log(pet.toLowerCase());

The result of the console.log will be "dog".

Chapter 22: Strings: measuring length and extracting parts

You are able to cut out portions of a string.

input: var seg = "unbelievable".slice(2,12);;

if you console.log seg you will get "believable".

Chapter 23: Strings: finding segments

This chapter goes over strings and how to pull out segments of a string.

var text = "The panthers could be the best team in the AFC South.";

var index = text.lastIndexOf("the");

console.log(index); returns 39

Chapter 24: Strings: finding a character at a location

This chapter talked about locating a character in a string

var text = "What is this";

alert(text.charAt(1));

This alert will just display an h

Chapter 25: Strings: replacing characters

You are able to replace characters and text using keywords like "replace"

var oldTeam = "Chiefs";

var newTeam = oldTeam.replace("Chiefs", "Raiders");

console.log(newTeam); will give you "Raiders".

Chapter 26: Rounding numbers

This chapter talks about rounding numbers up or down.

Math.ceil will round a number up and Math.floor will round a number down.

Chapter 27: Generating random numbers

This chapter goes over generating random numbers.

The Math.random phrase will generate a random number.

Chapter 28: Converting strings to integers and decimals

This chapter talks about converting strings to numbers and numbers to strings.

The parseFloat keyword will convert a string to a number and the parseInt will convert a number to a string.

Chapter 29: Converting strings to numbers, numbers to strings

This chapter talks about converting numbers to strings and strings to numbers. The syntax is very important to get this done.

If a variable notNum is a string, you can convert this to a number by using the keyword Number. So it will look like Number(notNum).

If a variable num is a number and we want to turn this into a string we will use num.toString(); to convert it to a number.

Chapter 30: Controlling the length of decimals

This chapter takes a deep dive into the toFixed keyword. This allows the user to control the decimal places, but also convert the number to a string.

An example of this is, var num = 1.234567;

var newNum = num.toFixed(1);

console.log(newNum);
